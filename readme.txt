* host *
Contains host code for each kernel (workload generation and preparation/collection of data to/from the FPGA)

* kernel * 
Contains hw kernels for different functions.

* kernel_config_files * 
Contains cfg files required for kernels (during compilation of kernel code, system synthesis and packaging)

* template_project *
Contains a template (gemm) project, that can be used to test other kernels too. Detailed instructions on what needs to be modified for other kernels are in its readme file.
